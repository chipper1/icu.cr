    Overview

    Currencies

    Encapsulates information about a currency

    Usage

    ICU::Currencies.currency("fr_FR")   # => "EUR"
ICU::Currencies.numeric_code("EUR") # => 978

    NOTE the .numeric_code method requires ICU >= 49
