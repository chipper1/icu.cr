    • ICU::StringSearch
    • Reference
    • Object

    Overview

    String Searching

    Provides language-sensitive text searching.

    Usage

    col = ICU::Collator.new("de_DE")
col.strength = ICU::Collator::Strength::Primary
search = ICU::StringSearch.new("ß", "...SS...", col)
search.next # => 3...5
